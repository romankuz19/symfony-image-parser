### Приложение "image-parser" на Symfony + Vue 3

Для запуска необходимо
1. Склонировать приложение из репозитория
2. Выполнить
````
composer install
````
3. Запустить приложение
````
symfony server:start -d
````
4. Перейти по адресу http://localhost:8000
